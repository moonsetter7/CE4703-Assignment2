Detailed Description

Provides interface for the entire card deck.

CardDeck will be implemented as a single linked list. This linked list uses dummy nodes for both the head and tail, containing no card data.

A "CardDeck" is a collection of 0,1 or more cards. This data type supports any number of packs of cards.
